# 🔥 Firewall Simulator

Firewall Simulator is a Java Swing based desktop application that simulates real-time network traffic monitoring and firewall rule management.

The application automatically generates random network packets and checks them against firewall rules to determine whether the traffic should be **allowed or blocked**. It provides a graphical user interface to visualize network activity, manage rules, and monitor traffic statistics.

---

## 🚀 Features

* Real-time network traffic simulation
* Firewall rule creation and deletion
* Traffic status display (Allowed / Blocked)
* Live statistics monitoring (Total, Allowed, Blocked traffic)
* Activity logging with timestamps
* User-friendly GUI built using Java Swing

---

## 🛠️ Technologies Used

* Java
* Java Swing (GUI Framework)
* Object Oriented Programming
* Event Driven Programming
* Timer-based Traffic Simulation

---

## 📌 Project Purpose

This project is developed to demonstrate the working concept of firewalls and network traffic filtering in an interactive graphical environment. It is useful for academic learning and visualization of networking concepts.

---

## ▶️ How to Run the Project

1. Download or clone the repository

2. Compile the Java file

   ```
   javac FirewallSimulator.java
   ```

3. Run the program

   ```
   java FirewallSimulator
   ```

---

## 📊 Functional Overview

* Random IP traffic is generated every few seconds
* Firewall rules are matched against destination IP and port
* Traffic is marked as **BLOCKED** or **ALLOWED**
* Statistics panel updates automatically
* Logs record important firewall activities
